Contact Form

The form is written in PHP and can be found in index.php. Be sure that your server supports PHP and has all the required settings for build-in mail() PHP function.

All you need to do to enable the PHP form is to point the user to the index.php instead of index.html and change the following variable in the ./incl/send-email.php to your actual email address:

<?php
  •  
  • /* set the email of the recipient (your email) */
  • $recipient = "test@example.com";

    • The message will be sent directly from your website to the email address you have specified above and will have the following form:

    ----- Info about the sender -----

Name: [name]
Email: [email]
Phone: [phone]


----- Message -----

[the actual message from the user which sent the email]
